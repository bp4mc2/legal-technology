# Legacy media-map

Deze map is gedeeltelijk uitgefaseerd.

## Let op

- Gegenereerde ReSpec documentatie (markdown/html/js includes) hoort niet meer in `media/`.
- De huidige bron van waarheid staat in:
  - `build/docs/*` voor build-artifacts
  - `dist/*.html` en `dist/assets` voor statische publicatie-output
  - `dist/pdf/index.pdf` voor PDF-output

Zie ook: `docs/migration-media-to-build-dist.md`.

## Nog wel in gebruik

Niet-ReSpec-artifacts zoals graphml/afbeeldingen kunnen hier tijdelijk blijven staan zolang scripts hiernaar verwijzen.
